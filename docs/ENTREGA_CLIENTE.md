# Entrega profesional al cliente

## Lo que recibe el cliente

- URL de acceso al dashboard.
- Usuario administrador.
- Página pública de reservas.
- QR personalizado.
- Servicios configurados.
- Barberos configurados.
- Horarios configurados.
- Guía rápida de uso.
- Soporte inicial.

## Checklist antes de entregar

- [ ] Login funciona.
- [ ] Barbería creada.
- [ ] Servicios configurados.
- [ ] Barberos configurados.
- [ ] Página pública visible.
- [ ] QR apunta a la página correcta.
- [ ] Se puede crear una cita.
- [ ] No se permiten dobles reservas.
- [ ] Dashboard muestra métricas.
- [ ] Pagos manuales funcionan.
- [ ] Dominio configurado.
- [ ] Variables de entorno en producción.
- [ ] Backup/revisión de Supabase.
