# Dashboard

## Objetivo

Que el dueño vea la salud diaria de la barbería.

## Métricas principales

- Citas de hoy.
- Ingresos de hoy.
- Próxima cita.
- Clientes nuevos.
- Citas canceladas.
- Servicios más vendidos.
- Barbero con más citas.

## Menú

- Dashboard.
- Agenda.
- Clientes.
- Servicios.
- Barberos.
- Pagos.
- Reservas públicas/QR.
- Configuración.

## Principio UX

El dashboard debe ser simple. El dueño de barbería no quiere un ERP complejo.

## Acción rápida

Debe haber botones para:

- Nueva cita.
- Nuevo cliente.
- Registrar pago.
- Ver QR.
